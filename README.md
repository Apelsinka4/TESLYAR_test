# TESLYAR Amazon EU Dashboard

Локальний статичний дашборд для аналізу Amazon P&L/PPC CSV. Менеджер відкриває сторінку, вибирає CSV-файли за потрібні місяці, а дашборд будується прямо у браузері. Дані не відправляються на сервер.

## Як користуватись

1. Відкрийте сайт або локальний `index.html`.
2. Натисніть **Вибрати CSV**.
3. Оберіть P&L-файли та PPC-файли за мінімум два місяці.
4. У селекторах виберіть поточний місяць і місяць для порівняння.

Для локального перегляду через сервер:

```bash
python -m http.server 8000
```

Після цього відкрийте [http://localhost:8000](http://localhost:8000).

## Що зроблено

- Огляд портфелю: Sales, Net profit, Margin з MoM-порівнянням.
- Розріз по країнах: сортування за зміною продажів, прибутку, маржі або PPC ACOS.
- Top-N по артикулах: продажі, прибуток, зростання прибутку або рейтинг уваги.
- AI-фіча: автокоментар “що змінилось” і watchlist ASIN/SKU, на які варто звернути увагу.
- Багаторазове оновлення: можна завантажувати новий набір CSV без зміни коду.

## Стек і чому так

Стек: vanilla HTML/CSS/JS без бекенду і залежностей. Для тестового це найпростіший шлях для нетехнічного користувача: достатньо відкрити сторінку і вибрати файли. Оскільки CSV парсяться у браузері, реальні фінансові дані не треба публікувати в репозиторій або відправляти на сервер.

Код обробляє `;` у P&L-файлах, європейську нотацію чисел з комою та пробілами, різний регістр і подвійні підкреслення у назвах файлів. PPC-звіти підтягуються на рівні країни для ACOS/ROAS.

## Що б додав далі

Додав би валідацію файлів перед імпортом, експорт PDF-summary, збереження локальної історії у браузері, повноцінний LLM Q&A по очищених даних і глибший join PPC до P&L на рівні ASIN/SKU.
